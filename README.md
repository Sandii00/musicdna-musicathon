# MusicDNA - AI-Powered Song Genealogy

## Overview
MusicDNA discovers the musical influences behind any song using lyrical analysis and live concert data.

## Features
- Song search via Musixmatch API
- Lyrical DNA extraction (AI analysis)
- Influence matching (similar song discovery)
- Interactive visualization
- Live tour date integration (JamBase)

## Tech Stack
- Frontend: React + D3.js
- Backend: Node.js + Express
- APIs: Musixmatch Pro, JamBase
